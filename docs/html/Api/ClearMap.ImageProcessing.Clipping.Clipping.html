Clipping¶

Clipping

Module to compute clipped images

Useful to save memory in large data sets

clip(source, sink=None, clip_min=None, clip_max=None, clip_norm=None, processes=None, verbose=False)[source]¶

Clip and normalize data.

Arguments

sourcearray

Input source.

sinkarray, dtype or None

output sink or output data type, if None, a new array is allocated.

clip_minnumber

Minimal number to clip source data to.

clip_maxnumber

Maximal number to clip source data to.

clip_normnumber

Normalization constant.

Returns

sinkarray

Clipped output.
